TQ Blacksmith V 0.1.3


Greetings all!

This tool is for those who have farmed for years in Titan Quest and haven't found some items. And for those who want to experiment with different builds based on specific gear...

This tool creates almost all the items in the game. Even all monster infrequents and quest items! Including common and broken items is very overwhelming and of less use, so these kinds of items and affixes have not been excluded in the tool.
You can also set prefix, suffix, bonus or seed to created item(including epic and legendary items as well). You don't have to visit Durin to upgrade epic and legendary gear.

The tool has been tested on Titan Quest Anniversary Edition v 1.54, 1.56 and 1.57.  Not sure about old versions.

Usage:
1) Create a new character in main quest in the game. And visit first caravan. It is preferred not to use existing characters.
2) Save and exit the character.
3) Open the TQ_Blacksmith.exe
4) Set game save path. e.g. C:\Users\<user>\Documents\My Games\Titan Quest - Immortal Throne
5) Set game installation path. e.g. F:\Games\Titan Quest154
6) Select the character. Make sure the character is not open in the game, TQVault or any other application.
7) Character's first sack will be shown. Where you can put created items.
8) Go to menu 'Smithy -> Create Item'. Creation wizard will pop up. Use Wizard to create an item.
9)New Item will be placed in the main stash.
10) To ease the implementation all created items are placed at (0,0) location in the stash. Inside the game you can unstack them.

11)'File->Save Changes' will save created items in character save file. Inside game you can put the items in Transfer Area to share with other characters. Also backup of original file will be created: BlaksmithBackup_Player.chr.
OR
You can save changes to other location by clicking menu 'File->Save to...'. It'll save Player.chr, which you can use to replace main save file to unload items later.
12) 'Smithy->Empty the sack' removes all items from sack. So make sure you've unloaded all the created items in game.


Tips/warnings:
1) You can set item seed on bottom right corner of 'Select Item' Page in creation wizard.
2) Having quest items in stash before time could break the quest. So be careful not have such items.
3) log.txt contains logs. if the application throws error check the log file.
4) detailLogs=True could be set in Configurations.txt file to have detailed logs.
5) If you face file access errors, try to run the application exe with admin privileges.


6) If you are not able to modify settings in application, then close the application and edit Configurations.txt.
